## ✨ 这是什么？

AnimStudio 是一个专为**独立动画创作者**打造的创作管理工具。

你是不是也这样：灵感散落在微信、备忘录、截图文件夹里，跟 AI 聊了一堆好点子但找不到了，故事改了八版不知道哪版好，分镜画了一半忘了节奏感，制作到一半不知道进度到哪了……

AnimStudio 就是来解决这些问题的。它把动画短片创作的**每一个环节**都装进了一个工具里，从你脑子里冒出第一个灵感，到最终输出成片。

## 🎯 功能全览

AnimStudio 一共有 **15 个功能模块**，按创作流程分为三大组：

### 🎨 创作组 — 把故事从脑子里掏出来

#### 📖 故事编辑器
不是一个大文本框，而是**结构化的卡片式编辑**：
- **Logline** — 一句话概括你的故事核心
- **世界观设定** — 时间、地点、规则、氛围
- **故事结构** — 按「幕」组织，每幕下面是可无限添加的 Beat（节拍点）
- 幕和 Beat 都支持拖拽排序、重命名、删除

> 💡 就像搭积木一样构建你的故事，每个 Beat 是一块积木，幕是一层楼。

#### 👤 角色管理
- 角色卡片：名字、身份（主角/反派/配角/导师…）、描述、性格、备注
- **头像上传** — 点击头像区域选择图片，自动压缩存储
- **角色关系图** — 添加角色之间的关系（师徒、敌对、暗恋……）

#### 💬 对白管理
- 按角色筛选对白
- 每条对白可指定角色、情绪（平静/开心/悲伤/愤怒/惊讶…共10种）
- 对白按顺序排列，方便梳理整体对话节奏

#### 🎬 分镜系统
每个镜头是一张**标准化卡片**，包含：
- **画面区域** — 点击上传草稿/参考图（自动压缩）
- **景别** — 远景 / 全景 / 中景 / 近景 / 特写 / 大特写
- **机位** — 平拍 / 俯拍 / 仰拍 / 斜拍 / 鸟瞰
- **运镜** — 固定 / 推进 / 拉远 / 摇移 / 跟拍 / 环绕
- **时长** — 预估秒数
- **动作描述 / 对白 / 音效备注** — 三个独立文本区

> 💡 顶部实时显示总镜头数和预估总时长。

#### 🗺️ 场景管理
- 创建场景卡片（名称、描述、备注）
- 关联镜头 — 勾选哪些分镜镜头发生在这个场景
- 编辑/查看模式切换

### ⚙️ 制作组 — 把故事变成动画

#### 📋 制作流程看板
覆盖动画短片制作的**完整 18 个环节**，分三大阶段：

| 前期 Pre-production | 中期 Production | 后期 Post-production |
|---|---|---|
| 创意/灵感收集 | 场景/背景制作 | 剪辑 |
| Logline | 资产制作 | 音效/配乐 |
| 故事大纲 | 骨骼绑定 | 视觉特效 |
| 剧本 | 关键帧动画 | 输出成片 |
| 世界观/设定 | 合成 | |
| 角色设计 | | |
| 概念艺术 | | |
| 分镜 | | |
| 动态分镜 | | |

每个环节可展开为独立工作区：
- **状态标记** — 未开始 / 进行中 / 已完成
- **笔记区** — 记录想法和进展
- **检查清单** — 自定义待办事项，逐项勾选

#### 📦 道具清单
- 添加道具（名称、描述、状态、关联场景）
- 按状态筛选：待处理 / 进行中 / 已完成
- 每个道具可关联到具体场景

#### 🎵 音频规划
双标签页设计：
- **配乐** — 时间段、风格、情绪、备注
- **音效** — 名称、描述、状态、关联场景

#### 🏁 里程碑追踪
- 添加里程碑（标题、截止日期、关联流程阶段、备注）
- 一键标记完成/未完成
- 按日期自动排序
- 顶部显示完成进度

---

### 📚 素材组 — 收集一切有用的东西

#### 🌐 网页抓取（爬虫）
这是 AnimStudio 的杀手级功能：

```
输入网址 → 点击「抓取」→ 自动提取标题、正文、图片 → 预览 → 一键导入参考库
```

- 输入任意网址，自动抓取页面内容
- 智能提取正文（过滤导航栏、广告、脚本等噪音）
- 发现页面中的所有图片，可点击选择想要的
- 一键导入到参考库，标题、链接、内容自动填充
- Web 模式通过 CORS 代理实现，Chrome 扩展模式无限制

> 💡 看到一篇好的角色设计教程？一个漂亮的色彩参考？直接粘贴网址，3 秒入库。

#### 🔖 参考库
- 按维度分类：构图 / 色彩 / 节奏 / 角色设计 / 场景设计 / 动画风格 / 分镜 / 光影 / 其他
- 每条参考：标题、链接（可直接打开）、分类、笔记
- 筛选器快速切换分类

#### 📝 便签
- 彩色便签卡片，随手记录灵感
- 支持关联到流程阶段

#### 📓 制作日志
每天一条日志，三个结构化字段：
- 📝 **今日工作** — 今天做了什么
- ⚠️ **遇到问题** — 卡在哪里了
- 🎯 **明日计划** — 明天打算做什么

> 💡 养成记录习惯，回头看的时候你会感谢自己。

#### 📊 统计面板
一眼看清项目全貌：
- 总镜头数 / 预估总时长
- 角色数 / 幕数 / Beat 数
- 场景数 / 对白条数 / 参考素材数
- 流程进度 / 道具进度 / 音效进度 / 里程碑完成率

## 🧩 Chrome 扩展模式

AnimStudio 同时也是一个 **Chrome Extension (Manifest V3)**，安装后解锁额外能力：

### 🤖 AI 对话抓取
在以下 AI 网站自动显示 🎬 浮动抓取按钮：
- **DeepSeek** (chat.deepseek.com)
- **Kimi** (kimi.moonshot.cn)
- **豆包** (www.doubao.com)
- **ChatGPT** (chatgpt.com)

点击按钮，一键抓取整个对话内容，导入到项目中。

支持三种导入模式：
- **全部导入** — 整个对话一键入库
- **选择性导入** — 预览对话列表，勾选想要的条目
- **划选导入** — 在 AI 网页上选中文字，右键收集

### 📎 右键收集
在任意网页上选中文字或图片 → 右键 → **「📎 收集到 AnimStudio」**

### 🌐 增强爬虫
扩展模式下网页抓取不受 CORS 限制，可以抓取任意网站。

## 🚀 快速开始

### Web 模式（开发/日常使用）

```bash
cd animation-helper
npm install
npm run dev
```

打开 `http://localhost:5173`，开始创作。

### Chrome 扩展模式（解锁 AI 抓取 + 右键收集）

```bash
cd animation-helper
npm run build
```

1. 打开 Chrome，访问 `chrome://extensions/`
2. 开启右上角「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择 `animation-helper/dist` 目录
5. 点击扩展图标，AnimStudio 在弹窗中打开
6. 访问 DeepSeek/Kimi 等 AI 网站，会自动出现 🎬 抓取按钮

## 💡 设计理念

1. **零成本 AI** — 不接任何 API，用免费的 DeepSeek/Kimi 聊故事，插件帮你整理
2. **纯本地存储** — 所有数据存在浏览器 IndexedDB 里，不上传任何服务器
3. **结构化创作** — 不是给你一个空白文档，而是用动画行业标准流程引导你
4. **渐进式使用** — Web 模式开箱即用，想要更多能力再装 Chrome 扩展
5. **创作者审美** — 深色沉浸界面，像深夜创作时打开的私人工作台
